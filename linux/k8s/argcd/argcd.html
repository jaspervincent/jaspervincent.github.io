kubekey 安装

cd ~
mkdir kubekey
cd kubekey
wget https://github.com/kubesphere/kubekey/releases/download/v2.0.0/kubekey-v2.0.0-linux-amd64.tar.gz
tar xf kubekey-v2.0.0-linux-amd64.tar.gz
# 仅保留 kk 二进行文件
./kk version
