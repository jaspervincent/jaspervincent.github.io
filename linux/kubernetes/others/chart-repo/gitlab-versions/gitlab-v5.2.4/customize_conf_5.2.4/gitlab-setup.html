3获取注册token

- 进入admin--概览--Runner
获取group类型的runnertoken
- 进入group -> Settings -> CI/CD -> Runners -> Group Runners
获取specific类型的runnertoken
- 进入具体的项目 -> Settings -> CI/CD -> Runners -> Specific Runners
