原生自带截图

Chrome

  • 按 F12 进入开发者控制台
  • 按 Ctrl + Shift + p 输入"截图"或者"full"，找到"截取完整尺寸的屏幕截图"或"Capture full size screenshot"
  • 回车后，自动下载到本地一张png图片
