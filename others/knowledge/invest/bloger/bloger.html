直播

没有人一开始就很厉害，要敢于开始。

点赞、关注、加入粉丝团

硬件

  1. 手机/三角架

    高清视频外接单反

  2. 声卡/麦克风/监听耳机

    声卡不仅有混响、变音、降噪等功能，还能播放各种活跃直播间氛围的音效

    麦克风建议采用灵敏度高的电容式麦克风

    监听耳机可以及时发现唱歌有没有跑调、声音有没有延迟

  3. 手机散热器
  4. 直播背景

壁画、盆栽、手办、玩偶等，背景布，人物背部加个补光灯

开播习惯

  • 建议一周至少稳定播三天以上，每次30分钟以上，每天如果能播2小时就算是小小的胜利了。
  • 不存在最好的时间段，如果最适合自己的时间段

  • 有自己的规划和节奏

    开场、才艺展示、PK、话题互动、结尾感谢

    PlainB 应对直播间人气的变化

自我介绍

我是谁，我是哪里人，我是干什么的，有没有代表作，我直播多久了，为什么做直播，直播内容是什么，每天都几点播等。重点突出自己的特色，让别人能够快速记住你。

下播

  • 复盘

  • 拓展直播社交圈

    去别人的直播间观摩与互动

  • 用心维护粉丝

提升直播间人气

  • 封面+标题+同城定位

    封面建议选择高清、背景简洁、符合自己内容定位无水印的本人照片

  • 找准适合自己的时间段

    直播前半小时左右发布短视频、并且坚持发布短视频积累粉丝

    直播前在粉丝群发布通知

  • 找准直播话题度

    找正向的话题而不是蹭热度

  • 参与官方活动

    获得dou+券、人气卡等奖励

其他开播方式

  • 电脑-直播伴侣

  • 外接单反相机

    准备：相机、采集卡、两根线(USB3.0, HDMI)，用USB3.0连接电脑和采集卡，相机专用HDMI线连接相机和采集卡

    相机 -> HDMI线 -> 采集卡 -> USB线 -> 电脑

    直播伴侣选择采集模式，视频采集信号中的自己采集器的名称，开始直播。

有没有被限流

创作者中心 –> 账号检测

如果限流，平台会发送违规通知

客服入口

共3个

  • 创作者底部，“如有问题，请联系客服”，点击“去反馈”
  • 主播中心，点击更多功能，找到“我的客服” 人工服务

互动技巧

观众不喜欢说话，破冰

  • 观察并发现有效信息

    头像、昵称、属地等寻找突破点

  • 直播间与新观众互动无回应
    • 可私息找招呼简单介绍或者邀请进入粉丝群
    • 红包、福袋鼓励公屏评论。“加油新主播”或其他口号
  • 根据新进直播间的数量和频率采取不同的互动方式
    • 新进来比较多时，可以用大家、你们、刚进来的朋友，

互动

  • 发福袋

    加入粉丝团，输入口令后参与

    抖币福袋不能加入实物口令。实物福袋10万粉丝后才能解锁。

  • 发红包

    封面会显示红包状态。人气高峰发红包更容易涨粉。

  • 话题

    热点话题，音乐榜

    公屏发言

    向粉丝征求话题

    看其他主播的话题

  • 玩什么？

    • 装扮道具类

      吸引粉丝注意：可爱的发箍、趣味的帽子、搞笑的玩具

    • 游戏类互动

      成语接龙、脑筋急转弯、猜歌名等

    • 模仿类互动

      卡通人物、木头人、大猩猩、表情包。

    • bgm互动

      在特定的场景或者面对特定的人群可定制专属的BGM.如PK时播放振奋人心的、节奏感十足的bgm; 感谢主题的歌曲；

主播的平均成长周期是60天。坚持

官方：新主播流量计划、新星成长训练营、闪耀吧星星、新主播社群

收入

星图

CPM 广告千次曝光成本(被1千人浏览到) = 视频报价 / 总曝光次数 * 1000

如视频报价: 5000元， 视频曝光：10W次，cpm = 5000 / 100,000 * 1000 = 50元

视频报价 = cpm * 曝光共次数 / 1000

cpm 越低被广告选择的机率越高
