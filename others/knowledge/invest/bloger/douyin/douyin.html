直播

没有人一开始就很厉害，要敢于开始。

点赞、关注、加入粉丝团

硬件

  1. 手机/三角架

    高清视频外接单反

  2. 声卡/麦克风/监听耳机

    声卡不仅有混响、变音、降噪等功能，还能播放各种活跃直播间氛围的音效

    麦克风建议采用灵敏度高的电容式麦克风

    监听耳机可以及时发现唱歌有没有跑调、声音有没有延迟

  3. 手机散热器
  4. 直播背景

壁画、盆栽、手办、玩偶等，背景布，人物背部加个补光灯

开播习惯

  • 建议一周至少稳定播三天以上，每次30分钟以上，每天如果能播2小时就算是小小的胜利了。
  • 不存在最好的时间段，如果最适合自己的时间段

  • 有自己的规划和节奏

    开场、才艺展示、PK、话题互动、结尾感谢

    PlainB 应对直播间人气的变化

自我介绍

我是谁，我是哪里人，我是干什么的，有没有代表作，我直播多久了，为什么做直播，直播内容是什么，每天都几点播等。重点突出自己的特色，让别人能够快速记住你。

收入

星图

CPM 广告千次曝光成本(被1千人浏览到) = 视频报价 / 总曝光次数 * 1000

如视频报价: 5000元， 视频曝光：10W次，cpm = 5000 / 100,000 * 1000 = 50元

视频报价 = cpm * 曝光共次数 / 1000

cpm 越低被广告选择的机率越高

首播流程

一、贴直播间内容标签三个方法

  1. 开播就设置精准标题(文字识别系统)
  2. 邀约老铁（从推荐进做5个动作点灯、打字、点赞、免费人气票、停留）
  3. 留精准粉

两大认知：对自我认知：了解自己适合什么风格（讲故事卖惨、新奇特、唱歌跳舞对口型、好好讲内容）； 对平台认知：三大识别系统（画面、语音、文字）

二、过三关（对自己的考验）

  1. 大头娃娃（是流量扶持，有状态自我介绍，语音识别系统能识别到你是来干啥的）
  2. 同城（同城一句话解决，你们在昆明哪里呀，你们知道普通人可以来开播吗？）
  3. 关注页精准粉：（留人才能推更多的人，1热情真诚 2存在感 3低姿态）

简单介绍自己（人设）、账号情况（多少粉丝开播的、有没有投抖加、有没有发作品、有没有邀约老铁、新账号还是老账号）、首播心得。

三、突破个位数（拉流量）拉流量前两件事

  1. 铺垫稳老铁（感恩、见证，传递价值和目标）
  2. 调整状态，拉流量，讲开始白

四、叠加流量

  1. 引共鸣
  2. 抛结果（拿出状态、强调首播、人数对比）
  3. 抛话题（三个话题）

五、给方法稳流量（三个话题）

  1. 来新人接流量讲自己
  2. 宣传抖音
    1. 什么样的人可以来做这种类型的主播
    2. 直播间的变现方式
    3. 抖音未来的趋势

1、我们这种门槛最低的主播不唱歌基本上有四类人来做：

第一个是宝爸宝妈，宝爷爷宝奶奶的，就是你们在家为了看孩子没有工作，没有经济收入的

第二个，如果你们是上班族，你对现在的这个收入不满意的，想改变生活现状的

第三个，如果你是负债的也可以投资时间来试一试

第四个，你是如果是老板的，是企业家的，也可以通过抖音这个平台宣传门产品，线上线下结合

2、咱们普通人来开播三种变现方式：音浪、带货、奖励

3、抖音未来的趋势：抖音正在从一个娱乐平台转向一个电商平台，未来是一个全民带货的时代，一个播商的时代，我跟大家一样一直在看热闹，但我现在参与进来了，我也想挣生活费，我也想带货。现在抖音注册用户14亿，日活用户8亿，流量很大，刷着玩的人很多，创业者很少，所以这是我们普通老百姓的机会。不需要颜值不需要才艺不需要学历不需要背景，只要满18岁，手里有一个抖音号谁都可以来开播。而且学习是免费的，开播也是免费的，不用花钱，唯一需要投资的就是时间。抖音24小时不打烊，无论你身在何处无论你身兼何职，不管你是宝爸宝妈还是上班族做生意的，只要每天能抽出2小时开播就行，非常自由。将来每个家庭都会有一个主播，咱一定要牢牢抓住机会。

起号、直播间进人三个方法

开播必备的两个条件

  1. 涨粉丝（1000个精准活跃粉）
  2. 实名认证

起号的一个步骤（没开播的宝宝一定要听，否则开播不进人。开播的个位数的听听是否开播前几步没做到位所以不进人）

  1. 定位（找对标账号学习）、贴标签：喜好标签、账号标签（手动）、直播间标签
  2. 活跃账号（静默账号抖音发现不了你）
    1. 发作品（垂直作品）
    2. 看直播（点灯、打字、点赞）

  3. 定一个有效的开播时间（上下播注意事项），有的人直播间不进人就是没有注意这几点

    上播三个注意事项

    1. 错开小时榜
    2. 选一个固定的时间开播
    3. 避开大主播上播时间

    下播三个注意事项

    1. 直播间进人的时间段不要下播
    2. 下播人数不要太低
    3. 非正常情况下播（说错话被限流立马下播）
  4. 准备直播大纲 设备.场地
  5. 宣传（1名字预告、2视频预告、3直播间曝光）

    恐播4怕：

    1. 怕没有人–开播前找流量、做好准备
    2. 怕没话说–准备内容。练口才
    3. 怕熟人笑话–屏蔽熟人。摆正心态（不要否认这份工作）
    4. 怕留不住人–引共鸣，抛话题、见证法、转化数据、画面留人
  3. 不要认为粉丝多，直播间人就多
  4. 不要等作品上热门再开播
  5. 为什么来抖音创业。家人是否支持理解

直播间六不要：

  1. 不要主要去要数据（引导、利他）
  2. 不要只欢迎和感谢
  3. 不要没有状态
  4. 不要大幅度的介绍自己
  5. 不要过度拉时长
  6. 不要和老铁闲聊
