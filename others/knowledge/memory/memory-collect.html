提高记忆

记忆：编码、提取

首先明确学习目的，其次调动多重感官建立知识联系，然后把握复习时机，最后主动回忆练习

  • 明确学习目的
    • 如读书前，先问问自己，我想从这本书里解决什么问题、学到什么方法。这样大脑就像导航一样，自动锁定关键信息。

提升记忆效率的方法：

  • 多重感官参与
    • 如背诵时，可以读出声音；学习时可以动手画思维导图，给重要的知识点配上特定的动作或者场景想象。
  • 知识结构化
    • 如果学习历史时，不要孤立的背诵事件年代，而是要理解事件之间的因果关系；
    • 学习数学时，不要机械记忆公式，而是要理解公式的推导过程和应用场景。

实用技巧

  • 预习
    • 学习新知识前，先花10分钟快速浏览整体框架，就像去陌生地址先看地图一样。
    • 看书时，先看目录和章节概要
    • 学习新课程前，先了解这门课的知识体系。
  • 做笔记
    • 比较好的是康耐尔笔记法，即把页面分成三个部分，右边记录主要内容，左边写关键词或问题，下面写总结。笔记不是抄写，而是用自己的语言重新组织内容。
  • 复习
    • 记忆是有遗忘曲线的，学完新知识后，要及时复习。
    • 复习时间：学完后立即复习一次，24小时后再复习一次，一周后第三次复习，一个月后第四次复习，经过这样四轮复习，知识就能进入长期记忆。
  • 主动回忆
    • 尝试回忆刚才学过的内容

提高记忆2

  • 睡前1小时，黄金记忆
    • 大脑在睡眠中是没有休息的，它在不断加工处理信息。而睡前1小时记忆内容会被大脑皮质不断的机械重复，起床之后再次巩固这部分内容。
    • 睡前1小时记什么？
      • 听音频加强记忆。如听英语单词、自己总结录音笔记
  • 多样化练习
    • 激发大脑的不同区域。如穿插不同的学科进行学习；同一门学科穿插不同的学习方式、学习内容。
      • 如1小时背单词，把记忆和检测两种不同的学习方式结合起来，配合番茄钟学习法，先听25分钟的单词音频，再用25分钟背诵单词，最后用25分钟进行听写
  • 躺在床上背。这时大脑供血足。
  • 当你怎么记都记不住：捂耳朵并大声朗读。
    • 原理：声明是直接通过骨骼传输到内耳，用手捂住耳朵朗读，对大脑刺激会更加强烈。同时大声朗读时，说话和听自己说话的“双重动作”有助于大脑存储信息，使之成为长期记忆。
  • 当你记东西很慢：左右手握拳。
    • 记忆时右手握拳90秒，回忆时左手握拳90秒。原理：紧握右拳能够启动左大脑的活动，而左手握拳能够启动右大脑的活动。右手握拳可能有助于记忆信息的编码，而左手有助于信息提取。
  • 当你背完很容易忘记：闭眼回忆
    • 背诵完10分钟内，闭眼回忆10-20秒。闭上眼睛能让人对正在回忆的事物建立起更清晰的影像。
  • 当你记东西容易混淆：手腕转动
    • 背东西时顺时针转动手腕，再逆时针转动手腕，重复三组。原理：促进大脑不同区域之间的信息交流和整合。手腕部位的触觉和短时记忆能力有关联。
  • 当你背书不能集中注意力：手指轻敲桌面
    • 食指和中指轻敲桌面，节奏保持稳定，持续敲击20次为一组。原理：刺激神经保护机制，激活大脑中的注意力相关区域。
  • 考前揉耳朵
    • 考前双手同时轻揉耳朵，直到微微变红，发烫即可。耳通肾肾通脑，揉耳朵就是给大脑开机。
  • 考前吃酸
    • 提神醒脑，刺激神经
  • 冷水刺激法
    • 考前用冷水冲洗手腕10秒，或者嘴里含一小块冰
  • 考前捂眼睛
    • 轻轻捂住眼睛，然后眼球向右转三圈，再向左转三圈，最后使劲睁大双眼答题。不仅可以缓解紧张情绪，还可以提高做题速度减少失误。
  • 3-2-1记忆法
    • 先阅读3次，再用自己的话表达2次，最后再写1次。
  • 睡觉记忆法
    • 用睡前1小时进行记忆，起床后20分钟立马巩固记忆此内容
  • 记忆宫殿法
    • 将记忆内容与实物关联，将实物与故事关联，将故事与熟悉的空间关联，用倒推法背出要记忆的内容
  • 音频记忆法
    • 把要记忆的内容录制为音频，用自己的声音录制效果更好，利用睡前和赶车等碎片化时间反复听此音频。
  • 思维导图记忆法
    • 1.整理出知识点结构，2.梳理出结构里的逻辑关系，3.往里填写具体内容，4.标注出重点和难点
  • 狮子记忆法
    • 饥饿时记忆会变强，边走边背书会增强记忆，略微寒冷会增强记忆
  • 间隔效应
    • 学习25分钟后，闭眼发呆10秒，这里大脑会加速回放正在学习的内容。
  • 主动检索
    • 学习新知识后，立即闭卷复述，想象在给小白讲解，及时标注卡壳/不会的地方，进行重点学习和巩固。
  • 交叉学习法
    • 交叉不同类型或难度的学习任务。比如背单词–>写论文–>做思维导图，强制大脑切换思维。
  • 检索式学习法
    • 先回想再背书，先做题再学习，整理好错题集
  • 费曼学习法
    • 用自己的语言描述你学的东西，并且通过不断查漏补缺，直到把这个内容讲得清楚准确。
  • 两分钟起跑法
    • 从大的学习任务中拆解出一个只需要两分钟完成的小任务，然后立即开始做这个任务
  • 番茄钟学习法
    • 以25分钟为一个学习单位，专注学习25分钟、休息5分钟、专注学习25分钟这样循环。
  • 非惯用手练习
    • 每天5分钟非惯用手练习。左手写字，强制激活大脑
  • 交替学习法
    • 每学25分钟切换不同学科，每次切换前闭眼复述刚才的学习重点。大脑保持新鲜感，记忆留存率直接翻倍。

调整学习顺序

  • 先把书本合上，努力回想相关知识点，想不起来时千万别马上翻书，回想得越吃力越艰难，你就会记得越长久牢固。
  • 并且学习时要倒着来，不要反复阅读和背诵，要多回想多测试。比如做题、做试卷、做错题集。
  • 做错题集时，先抄题目，听老师或看完解析后，一定要隔上半天或一天再做一刻错题。卡壳或不会的地方重点标注，有多种解法也一直写上，再在错题集上做一个专栏专门记易错点。

养脑

  • 左手刷牙
    • 强制激活对策大脑半球
  • 每坐30分钟起身走2分钟
    • 久坐大脑会缺氧，脑子沉、注意力不集中，走两步就是给大脑开窗通风
  • 每天发呆5分钟
    • 促进记忆巩固，也容易产生灵感
  • 学习前运动10分钟
    • 运动进大脑会分泌帮助记忆的物质(BDNF)，运动后记得快又牢，还能更快理清逻辑
  • 15分钟内喝两杯水
    • 缺水会导致注意力和执行力下降，15分钟内两杯水(500ml)就能快速改善大脑认知表现，让思维更清晰。

专注力

  • 舒尔特方格法
    • 在纸上画25个方格，将1-25打乱填进去，再以最快的速度从1数到25，边数边指出数字。每天练习一次，持续两周，注意力提升40%。
  • 15分钟晨间启动术
    • 起床后立即做3件小事。比如背单词、写日程、整理书桌。低门槛任务激活大脑执行中枢，骗大脑进入学霸模式。全天专注时长增长2.7倍。

照相记忆

  • 第一遍先看，第二遍脑中想，第三遍再看一遍纠正错误。这样能在脑中呈现出图像，背的时候照着图像念就可以了。

训练照相记忆

  • 黄卡训练：主要用于激活残像记忆和心像能力。
  • 形状颜色训练：几何图形+色彩变化，刺激大脑视频记忆中枢
  • 曼陀罗图训练：通过高对称+复杂色彩图案刺激视觉系统。
  • 3D图训练：包括立体图、随机点立方体图、三维几何图形等，通过深度视觉刺激大脑空间感知和图像记忆中枢。
